## Design notes

- **Goal**: calm, friendly, Notion-like layout with soft pastels (“Sunrise Calm”).
- **Must include**:
  - Light + dark mode states
  - Hover/focus/disabled states for buttons and inputs
  - Spacing rules (consistent paddings, max widths, card radii)
