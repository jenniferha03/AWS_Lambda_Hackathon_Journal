# Page Design Doc — Landing

## Goal

Explain the product in a calm, friendly way and drive users to **Sign up**.

## Primary actions

- Start Free (go to signup)
- View Pricing

## Layout / sections

- Hero section with “Sunrise Calm” visuals + value proposition
- Product highlights (AI insights, habit building, analytics)
- Nature-inspired section (healing aesthetic)
- Footer (navigation + contact)

## States

- N/A (static content)

## Data

- **Reads**: none
- **Writes**: none
- **Calls**: none

## Components used

- `PublicLayout`
- Standard buttons, cards, sections

## Accessibility / UX notes

- Large readable typography, high contrast text
- Buttons must have clear hover/focus styles in both themes

## Acceptance criteria

- Hero fills width cleanly and renders well on mobile
- CTA buttons are visible in both light/dark mode
- Footer links are readable in both modes
